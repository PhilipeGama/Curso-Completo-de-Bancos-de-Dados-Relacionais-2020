Dicionáario de dados - Introdução

No dicionário de daos são armazenados os metadados, ou seja, os dados sobre os dados.

- Modelagem
  Levantamento, definição e registro dos dados em um arquivo ou planilha para auxília na modelagem
  conceitual, lógica e física do banco de dados.
  
 - SGBD
    Consulta dos metadados dos objetos do banco de dados.
